# CSS Styling Project

This project demonstrates CSS styling techniques for various HTML elements as specified in the assignment instructions.

## Project Structure

- `index.html` - Home page
- `about.html` - About page
- `contact.html` - Contact page
- `style.css` - Single stylesheet for all pages

## CSS Requirements Implemented

1. Single stylesheet for all three pages ✓
2. CSS rule for body tag (font family and size) ✓
3. CSS rule for header tag (background color) ✓
4. CSS rule for raw tag (background color) ✓
5. CSS rule for main tag (background color and font size) ✓
6. CSS rule for footer tag (background color) ✓
7. CSS rule for lt tag (display and width) ✓
8. CSS rule for ht tag (text alignment, font family, and color) ✓
9. CSS rule for p tag (line height) ✓
